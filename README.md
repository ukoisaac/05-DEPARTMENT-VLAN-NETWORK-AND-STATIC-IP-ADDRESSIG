# 05-DEPARTMENT-VLAN-NETWORK-AND-STATIC-IP-ADDRESSIG
## Designed and implemented VLAN-based segementation for departmental isolation in a simulated enterprise network 
## Skills Demonstrated

* Designed and implemented VLAN-based network segmentation to logically separate departmental traffic (Admin, HR, IT) within a Layer 2 switching environment.
Configured and managed VLANs on a Cisco switch, including VLAN creation, naming, and port assignment using Cisco IOS CLI.
Applied access port configuration to assign end devices to their respective VLANs, ensuring proper traffic isolation and broadcast domain control.
Developed an IP addressing scheme aligned with VLAN segmentation, assigning unique subnets to each department in accordance with best practices.
Verified and validated VLAN configurations using operational commands such as show vlan brief and show running-config.
Performed connectivity testing and troubleshooting to confirm intra-VLAN communication and inter-VLAN isolation using ICMP (ping).
Demonstrated practical understanding of Layer 2 switching concepts, including VLAN membership, broadcast containment, and traffic segmentation.
Utilized structured network design principles to model a small enterprise environment with department-based segmentation.
Gained hands-on experience with Cisco IOS command-line interface (CLI) for configuration, verification, and troubleshooting tasks.



### Real-World Application
This project models a typical enterprise network where departments are segmented using VLANs to enhance security, reduce broadcast traffic, and improve overall network performance. Such configurations are widely implemented in corporate environments to enforce logical separation of organizational units while maintaining a shared physical infrastructure.
